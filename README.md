# Layoutmaster.app

A free, open-source touch typing website.

> [!WARNING]
> This is a work in progress rewrite of the old LayoutMaster, some features are still missing. You can find the old version in the [legacy branch](https://github.com/GAMIS65/LayoutMaster/tree/legacy).
